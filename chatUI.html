Yeah, I ran into that too. You need to include the tenant ID in the header now, it's not optional anymore. Here is the snippet I used to fix it:
auth_helper.py 
import requests
def authenticate_user(tenant_id):
    # Add tenant ID to headers
    headers = {
        'X-Tenant-ID': tenant_id,
        'Authorization': 'Bearer ...'
    }
    return requests.get(url, headers=headers)
